Application Development Kit.
This process should be ran to get you started with some basic code and setup.

1. Run setup to create a apk.conf file
2. Run 9TD deploy to create file structure
3. Write database tables
4. Run database sql to create db and tables
5. Run describe sql to get db.description
6. Run daogenerator.sh to make code
7. Run publishdaocode.sh to move to src
8. Start developing
9. Deploy
